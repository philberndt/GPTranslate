GPTranslate

Fast, Modern Desktop Translation

A powerful AI powered desktop translation application built with Rust and Tauri that provides instant translation between multiple languages with global hotkey support, system tray integration, and a beautiful user interface.

Download for Windows View on GitHub

Why Choose GPTranslate?

Lightning Fast

Real-time translation as you type with 500ms debouncing for optimal performance and minimal API costs.

Global Hotkey

Capture and translate clipboard content from anywhere with configurable keyboard shortcuts.

AI Powered

Supports OpenAI and Azure OpenAI with intelligent language detection and custom prompts.

Modern UI

Beautiful, responsive interface with auto, light, and dark themes that adapt to your system.

Cost Effective

Extremely economical with gpt-4.1-nano model - less than $0.0001 per translation.

Free and Open Source

Completely free to use and modify, with source code available on GitHub under the MIT license.
