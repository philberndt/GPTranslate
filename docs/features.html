Features

Discover all the powerful capabilities that make GPTranslate the perfect translation companion

Core Translation Features

Intelligent Language Detection

GPTranslate automatically detects the source language from most supported languages, eliminating the need to manually specify the input language. The AI-powered detection is highly accurate and works with:

  • Over 100 languages and dialects
  • Mixed-language content
  • Technical and specialized terminology
  • Short phrases and long documents

Multi-language Translation

Seamless translation between dozens of languages with configurable target language preferences:

  • Primary Target Language: Your main translation language
  • Alternative Target Language: Fallback when source equals target
  • Custom Language Pairs: Define specific translation preferences
  • Context-Aware Translation: Maintains meaning across different language families

Real-time Translation

Experience lightning-fast translations with optimized performance:

  • 500ms Debouncing: Prevents excessive API calls while typing
  • Streaming Results: See translations appear as they're generated
  • Smart Caching: Reuses recent translations for better performance
  • Offline Fallback: Historical translations available without internet

Multiple AI Providers

Choose from various AI providers based on your needs and preferences:

OpenAI Integration

  • gpt-4.1-nano: Ultra-fast, cost-effective (recommended)
  • gpt-4.1-mini: Balanced performance and cost
  • gpt-4o-mini: Enhanced accuracy for complex texts
  • gpt-4.1: Maximum quality for professional use
  • gpt-4o: Latest model with advanced capabilities

Azure OpenAI Integration

  • Enterprise-grade security and compliance
  • Dedicated deployment models
  • Regional data residency options
  • Enhanced SLA and support

Ollama Integration (Local AI)

  • Complete Privacy: No data leaves your machine
  • No API Costs: Run translations locally
  • Offline Capability: Works without internet connection
  • Recommended Models: phi3-mini-128k-instruct, mistral-medium, qwen2

Custom Translation Prompts

Tailor the AI's behavior to your specific needs with powerful prompt customization:

  • Variable Support: Use {detected_language} and {target_language}
  • Style Control: Formal, casual, technical, creative translations
  • Context Instructions: Specify domain knowledge (medical, legal, technical)
  • Tone Preservation: Maintain the original text's emotional tone

User Interface Features

Modern, Responsive Design

Beautiful interface that adapts to your workflow:

  • Two-Panel Layout: Clear separation of input and output
  • Responsive Scaling: Works on any screen size
  • Accessibility: Full keyboard navigation and screen reader support
  • Intuitive Controls: Minimal learning curve

Theme Support

Customize the appearance to match your preferences:

  • Auto Theme: Follows system dark/light mode
  • Light Theme: Clean, bright interface
  • Dark Theme: Easy on the eyes for extended use
  • System Integration: Automatic theme switching with OS changes

Powerful Keyboard Shortcuts

Streamline your workflow with comprehensive keyboard support:

Global Hotkey (Configurable - Default: Ctrl+Alt+C)

  • When app not focused: Clipboard capture and translation
  • When app focused: Reset detected language
  • Always: Reset detected language first
  • Customizable: Set any key combination you prefer

Application Shortcuts

  • Ctrl+C: Copy translated text (when app focused)
  • Esc: Close settings/history dialogs
  • Tab: Navigate between interface elements
  • Enter: Trigger translation manually

System Integration Features

System Tray Integration

GPTranslate runs quietly in the background with full system tray support:

  • Always Available: Click tray icon to show/hide window
  • Context Menu: Right-click for quick actions
  • Status Indicators: Visual feedback for translation state
  • Minimize to Tray: Close button minimizes instead of exiting

Global Hotkey System

Capture and translate content from anywhere on your system:

  • Instant Access: No need to switch to the app
  • Clipboard Integration: Works with any copied text
  • Background Processing: Translations happen behind the scenes
  • Smart Activation: Only activates when needed

Auto-start with Windows

Seamless integration with your startup routine:

  • Boot Integration: Starts automatically with Windows
  • Fast Startup: Minimal impact on boot time
  • Background Launch: Starts minimized to tray
  • Optional: Can be disabled in settings

Single Instance Protection

Prevents conflicts and resource waste:

  • No Duplicates: Only one instance can run at a time
  • Smart Activation: Subsequent launches activate existing instance
  • Resource Efficiency: Minimal memory and CPU usage

Seamless Clipboard Operations

Intelligent clipboard handling for smooth workflow:

  • Smart Detection: Recognizes when new content is copied
  • Auto-paste: Results automatically copied to clipboard
  • Format Preservation: Maintains text formatting when possible
  • Error Handling: Graceful handling of clipboard conflicts

Data Management Features

Translation History

Keep track of all your translations with comprehensive history management:

  • Persistent Storage: All translations saved locally
  • Search Functionality: Find past translations quickly
  • Metadata Tracking: Source/target languages, timestamps
  • Export Options: Save history in various formats

JSON-based Configuration

Flexible settings management with full control:

  • Location: ~/.gptranslate/config.json
  • Human-readable: Easy to edit and backup
  • Version Control: Track configuration changes
  • Portable: Easy to transfer between machines

User-friendly Settings Interface

Comprehensive configuration without complexity:

  • Organized Sections: Logical grouping of related settings
  • Real-time Validation: Immediate feedback on configuration changes
  • Help Text: Contextual guidance for each setting
  • Reset Options: Restore defaults or individual settings

Performance & Reliability Features

Optimized Performance

Built for speed and efficiency:

  • Fast Startup: App launches in under 2 seconds
  • Low Memory Usage: Typically under 50MB RAM
  • Efficient Networking: Optimized API calls and caching
  • Background Processing: Non-blocking translation operations

Smart Request Management

Intelligent handling of translation requests:

  • Debouncing: Prevents excessive API calls while typing
  • Duplicate Detection: Avoids redundant translations
  • Request Queuing: Manages multiple simultaneous requests
  • Timeout Handling: Graceful handling of network issues

Comprehensive Error Handling

Robust error management for reliable operation:

  • Network Error Recovery: Automatic retry with exponential backoff
  • API Error Handling: Clear error messages and suggested fixes
  • Graceful Degradation: Fallback options when services are unavailable
  • Detailed Logging: Comprehensive logs for troubleshooting

Security & Privacy

Your data and privacy are protected:

  • Local Storage: All personal data stored on your machine
  • Secure API Communication: Encrypted connections to AI providers
  • No Telemetry: No usage data sent to developers
  • API Key Protection: Secure storage of sensitive credentials

Developer & Advanced Features

Hot Reload Development

Streamlined development experience:

  • Instant Updates: Changes reflected immediately
  • Vite Integration: Fast build system with HMR
  • Tauri Dev Server: Native app with web dev tools
  • Cross-platform Development: Develop on Windows, deploy anywhere

Extensive Logging

Comprehensive debugging and monitoring:

  • Console Logging: Real-time debug information
  • File Logging: Persistent logs for issue diagnosis
  • Configurable Levels: Debug, info, warn, error levels
  • Performance Metrics: Translation timing and success rates

Modular Architecture

Clean, maintainable codebase:

  • Separation of Concerns: Clear module boundaries
  • Plugin Architecture: Easy to extend with new providers
  • Type Safety: TypeScript and Rust type systems
  • Test Coverage: Comprehensive automated testing

Cost Efficiency Features

Optimized API Usage

Minimize costs while maximizing quality:

  • Smart Caching: Reuse translations for identical requests
  • Efficient Prompts: Optimized to minimize token usage
  • Model Selection: Choose cost-effective models like gpt-4.1-nano
  • Batch Processing: Combine multiple requests when beneficial

Usage Monitoring

Track and control your translation costs:

  • Token Counting: Real-time token usage display
  • Cost Estimation: Approximate costs for each translation
  • Usage Statistics: Daily, weekly, monthly usage reports
  • Budget Alerts: Notifications when approaching limits

Accessibility Features

Full Keyboard Support

Complete application control without a mouse:

  • Tab Navigation: Move between all interface elements
  • Keyboard Shortcuts: All major functions accessible via keyboard
  • Focus Indicators: Clear visual focus indicators
  • Screen Reader Support: Proper ARIA labels and semantics

Visual Accessibility

Designed for users with visual impairments:

  • Scalable Text: Respects system font size settings
  • Clear Typography: Readable fonts with good spacing
