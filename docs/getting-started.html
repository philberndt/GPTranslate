Getting Started

Everything you need to start translating with GPTranslate

Quick Start

Get up and running with GPTranslate in just a few minutes. This guide will walk you through downloading, installing, and configuring the application for your first translation.

Prerequisites

Before installing GPTranslate, ensure your system meets the following requirements:

System Requirements

  • Operating System: Windows 10 or Windows 11 (tested on Windows 11)
  • WebView2: Usually pre-installed on Windows 11, automatic download on Windows 10
  • Memory: Minimum 4GB RAM recommended
  • Storage: ~50MB available disk space

API Requirements

You'll need an API key from one of the supported providers:

  • OpenAI: API key from OpenAI Platform
  • Azure OpenAI: API key and endpoint from Azure Portal
  • Ollama: Local Ollama installation (optional)

Installation

Option 1: Download Pre-built Installer (Recommended)

  1. Visit the Releases page
  2. Download the latest Windows installer (.msi file)
  3. Run the installer and follow the setup wizard
  4. Launch GPTranslate from the Start menu or desktop shortcut

Option 2: Build from Source

For developers or advanced users who want to build from source:

Prerequisites for Building

  • Rust: 1.87+ with stable toolchain
  • Node.js: 22+ with npm
  • Visual Studio Build Tools: 2022 (Windows)

Build Steps

# Clone the repository
git clone https://github.com/philberndt/gptranslate.git
cd gptranslate

# Install dependencies
npm install

# Run in development mode
npm run tauri dev

# Or build for production
npm run tauri build

First-Time Setup

1. Initial Launch

When you first launch GPTranslate:

  • The application will appear in your system tray
  • Click the tray icon to open the main window
  • You'll see the translation interface with empty text areas

2. Configure API Settings

Before you can translate, you need to configure your API settings:

  1. Click the Settings icon (gear) at the bottom of the window
  2. Select your preferred provider (OpenAI, Azure OpenAI, or Ollama)
  3. Enter your API credentials
  4. Choose your preferred model
  5. Set your target language
  6. Click Save to apply settings

3. Set Up Global Hotkey

Configure the global hotkey for quick translations:

  1. In Settings, find the Hotkey field
  2. Click in the field and press your desired key combination
  3. Default is Ctrl+Alt+C - change if needed
  4. Save your settings

Your First Translation

Method 1: Direct Input

  1. Open the GPTranslate window
  2. Type or paste text in the left panel
  3. Translation appears automatically in the right panel
  4. Click Copy to copy the translation

Method 2: Global Hotkey

  1. Copy any text with Ctrl+C
  2. Press your global hotkey (default: Ctrl+Alt+C)
  3. GPTranslate opens with the translation ready
  4. Translation is automatically copied to clipboard

API Provider Setup

OpenAI Setup

  1. Go to OpenAI Platform
  2. Create an account or sign in
  3. Navigate to API Keys section
  4. Click "Create new secret key"
  5. Copy the API key
  6. In GPTranslate Settings:
    • Select "OpenAI" as provider
    • Enter your API key
    • Set model to gpt-4.1-nano (recommended for cost)

Azure OpenAI Setup

  1. Go to Azure Portal
  2. Create an Azure OpenAI resource
  3. Deploy a model (e.g., gpt-4.1-nano)
  4. Note your endpoint and API key
  5. In GPTranslate Settings:
    • Select "Azure OpenAI" as provider
    • Enter your endpoint URL
    • Enter your API key
    • Enter your deployment name

Ollama Setup (Advanced)

  1. Install Ollama locally
  2. Pull a translation model: 
    ollama pull phi3-mini-128k-instruct
  3. In GPTranslate Settings:
    • Select "Ollama" as provider
    • Set endpoint to http://localhost:11434
    • Enter model name (e.g., phi3-mini-128k-instruct)

Basic Usage

Translation Workflow

  1. Input Text: Type or paste text in the left panel
  2. Auto-Detection: Language is automatically detected
  3. Translation: Appears in the right panel after 500ms delay
  4. Copy Result: Use the Copy button or Ctrl+C

Keyboard Shortcuts

  • Global Hotkey (default Ctrl+Alt+C):
    • When app not focused: Capture clipboard and translate
    • When app focused: Reset detected language
  • Ctrl+C: Copy translated text (when app focused)
  • Esc: Close dialogs

System Tray Integration

  • Click tray icon: Show/hide main window
  • Right-click tray icon: Show context menu
  • Close button: Minimizes to tray (doesn't exit)

Troubleshooting

Common Issues

Translation Not Working

  • Check your API key is correct and valid
  • Verify internet connection
  • Ensure you have sufficient API credits/quota
  • Check the model name is correct for your provider

Global Hotkey Not Working

  • Try a different key combination
  • Check if another application is using the same hotkey
  • Run GPTranslate as administrator (if needed)
  • Restart the application after changing hotkey

App Not Starting

  • Ensure WebView2 is installed (download from Microsoft)
  • Check Windows version compatibility
  • Try running as administrator
  • Check antivirus software isn't blocking the app
